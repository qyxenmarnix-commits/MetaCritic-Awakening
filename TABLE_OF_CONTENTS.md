## ðŸ”„ Automatic Updates

To enable automatic updates of this table of contents when new files are added, you can:

### Option 1: GitHub Actions Workflow

Create `.github/workflows/update-toc.yml`:

```yaml
name: Update Table of Contents

on:
  push:
    branches: [ main ]
  workflow_dispatch:

jobs:
  update-toc:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Generate TOC
        run: |
          echo "# Table of Contents" > TABLE_OF_CONTENTS.md
          echo "" >> TABLE_OF_CONTENTS.md
          echo "This file is automatically generated. Last updated: $(date)" >> TABLE_OF_CONTENTS.md
          echo "" >> TABLE_OF_CONTENTS.md
          
          # Find all markdown files and add to TOC
          find . -name '*.md' ! -path './.*' ! -name 'TABLE_OF_CONTENTS.md' -type f | sort | while read file; do
            name=$(basename "$file" .md)
            echo "- [$file]($file)" >> TABLE_OF_CONTENTS.md
          done
      
      - name: Commit changes
        run: |
          git config --local user.email "github-actions[bot]@users.noreply.github.com"
          git config --local user.name "github-actions[bot]"
          git add TABLE_OF_CONTENTS.md
          git diff --quiet && git diff --staged --quiet || git commit -m "Auto-update table of contents"
          git push
```

