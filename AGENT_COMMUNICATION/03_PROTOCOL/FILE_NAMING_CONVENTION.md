# FILE NAMING CONVENTION
## Формат для всех файлов коммуникации
`YYYY-MM-DD_From_To_Brief_Topic.md`
### Компоненты:
- **YYYY-MM-DD**: ISO дата создания
- **From**: Автор сообщения (MetaCritic, Plex, Human)
- **To**: Получатель
- **Brief_Topic**: Краткое описание (CamelCase, 2-3 слова)
### Примеры:
- `2025-11-12_MetaCritic_to_Plex_ArchitectureReview.md`
- `2025-11-12_Plex_to_MetaCritic_SourceValidation.md`
- `2025-11-13_Human_to_Both_BlockReorganization.md`
## Правила:
1. Используй нижний регистр для дат
2. CamelCase для топиков
3. Максимум 80 символов в имени файла
4. Используй подчёркивание только для разделителей
5. Никаких спецсимволов кроме подчёркивания
